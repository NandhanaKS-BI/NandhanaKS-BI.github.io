# Nandhana K.S- Data Analysis Portfolio



## About This Portfolio Template

This portfolio is designed to demonstrate skills in:
- Data visualization and storytelling
- Data modeling and analysis
- Creating interactive, professional dashboards
- Transforming complex data into actionable insights

## 🚀 Projects


### 1. Sales Performance Analysis Dashboard (Excel)

Overview

In this project, I created a Sales Performance Dashboard using Excel. The dashboard tracks key business metrics such as total sales, total profit, and sales trends using Pivot Tables and KPI-based visualizations. It helps to monitor sales performance and identify key areas for improvement.

Tools Used

Microsoft Excel (Pivot Tables, Advanced Formulas, Data Analysis,visualizations)

### 2. Relational Database and Dashboard Preparation (SQL + Power BI)
Overview

This project involved the creation of a relational database using SQL, followed by building an interactive Power BI dashboard. The dashboard tracks sales, budget performance, and profit trends. The goal was to clean, structure, and visualize data to help business teams make informed decisions.

Tools Used

SQL (Data cleaning, Data Modeling, Queries)

Power BI (Dashboard Development, Data Visualization)

### 3. COVID-19 Global Analysis Dashboard (Tableau)
Overview

This Tableau project focuses on analyzing global COVID-19 trends. The interactive dashboard visualizes confirmed cases, deaths, recoveries, and the active cases across countries and regions. It also features KPIs and regional analysis to track the pandemic's impact.

Tools Used

Tableau (Dashboard Development, Visualizations)

Global COVID-19 dataset

### 4. Netflix Content Analysis Dashboard (Tableau)
Overview

In this project, I developed a Netflix Content Analysis Dashboard using Tableau. The dashboard visualizes Netflix's content library, including the distribution of movies, TV shows, and genres. It also highlights the top 10 countries, directors, and actors contributing to the platform’s content.

Tools Used

Tableau (Dashboard Development, Data Visualization)

Netflix dataset

### 5. Retail Sales Data with Seasonal Trends Analysis (Python + Power BI)
Overview

In this project, I used Python for data cleaning and analysis, followed by creating an interactive Power BI dashboard. The dashboard analyzes retail sales and warehouse sales, and includes seasonal trend analysis to help businesses identify demand patterns over time.

Tools Used

Python (Pandas, NumPy, Matplotlib, Seaborn for Data Analysis and Plotting)

Power BI (Dashboard Development)

### 6. HR Analytics Dashboard (Power BI)
Overview

This Power BI project focuses on analyzing employee attrition and performance metrics. The dashboard helps HR teams gain insights into the factors influencing employee retention and identifies top-performing departments.

Tools Used

Power BI (Dashboard Development, Data Visualization)

HR dataset (for analyzing employee data)

Skills & Tools Used:

Data Analysis & Visualization: SQL, Power BI, Excel, Tableau

Programming: Python (Pandas, NumPy, Matplotlib, Seaborn)

Business Intelligence: Dashboard Creation, KPI Reporting, Data Cleaning, Exploratory Data Analysis
